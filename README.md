# Plover Retro Quotes

Add single or double quotes to your words after you've already written them.

## Usage

Create some dictionary entries to trigger the retroactive quotes:

```
"KWO*ETS": "{:retro_double_quotes:1}",
"KWO*ETS/KWO*ETS": "{:retro_double_quotes:2}",
"KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:3}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:4}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:5}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:6}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:7}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:8}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:9}",
"KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS/KWO*ETS": "{:retro_double_quotes:10}",
"SKWO*ETS": "{:retro_single_quotes:1}",
"SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:2}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:3}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:4}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:5}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:6}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:7}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:8}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:9}",
"SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS/SKWO*ETS": "{:retro_single_quotes:10}",
```

Then you can use the strokes like this:

```
HEL/LOE/WORLD/KWO*ETS
hello "world"

HEL/LOE/WORLD/KWO*ETS/KWO*ETS
"hello world"

HEL/LOE/WORLD/SKWO*ETS
hello 'world'

HEL/LOE/WORLD/SKWO*ETS/SKWO*ETS
'hello world'
```
